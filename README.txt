
Step 1 : Getting a polymer starter-element running

Install Chrome
Install Web Server for Chrome from https://chrome.google.com/webstore/detail/web-server-for-chrome/ofhbbkphhbklhfoeikjpcbhemlocgigb

Download npm from https://nodejs.org/en/download/
Download git from https://git-scm.com/download/win
Install both
Restart machine

cmd
> cd WebContent
> npm install
> npm install git -g
> npm install -g bower
> bower install

Run the website from WebContent using Web Server for Chrome
Navigate to http://127.0.0.1:8887/elements/starter-element/demo/
